HTTP
Protocolo de Transferencia de Hipertexto (Hypertext Transfer Protocol): es el protocolo utilizado para la comunicación entre navegadores y servidores web.
HTML
Lenguaje de Marcado de Hipertexto (HyperText Markup Language): es el lenguaje estándar para la creación de páginas web.
URL
Localizador Uniforme de Recursos (Uniform Resource Locator): es la dirección única que identifica un recurso en la web.
TCP/IP
Conjunto de protocolos de comunicación que permite la interconexión de dispositivos en redes y la transmisión de datos en Internet.
Internet
Red global de redes interconectadas que permite el intercambio de información entre dispositivos en todo el mundo.
W3C
Consorcio World Wide Web (World Wide Web Consortium): organismo internacional que desarrolla estándares web para garantizar la accesibilidad y compatibilidad de la web.
Mozilla Foundation
Organización sin fines de lucro que promueve la apertura, la innovación y la participación en la web. Es conocida por su navegador web Firefox.
